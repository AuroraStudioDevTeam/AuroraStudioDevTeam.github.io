    ZPS革命日报¶

    编者：极光

    二月革命(2021.2)¶

    二五事变(2021.2.5)¶

    1. 旋风正式宣布革命分子为疯子
    2. 风疯私信旋风关于紫灵GHS的事，旋风回复:紫灵已经疯了，旋风和紫灵的关系进一步紧张

    3. 革命狂热分子白朴直接与旋风对线

      白朴:旋风是个什么东西 旋风:滚

    4. Aurora Studio 成立(第一个革命根据地)

      Aurora Studio简称ASU，是由原ZPS的后台部，后期部，编剧部剥离出来合并的工作室

    分析：二五事变成功让紫灵，白朴，羽酱等对ZPS有极大意见的核心人员提出起义想法

    二六争端(2021.2.6)¶

    1. 白朴使用大量命令方块引起旋风不满，后者与白朴关系进一步恶化
    2. 旋风警告紫灵停止革命运动 旋风原句:"紫灵咱能正常点么"

    分析：二六争端恶化了反革命者与革命者的关系，为后来的决裂埋下种子

    ASU/ZPS嘴炮战争(2021.2.8)¶

    1. ASU派伽赴ZPS主群抨击ZPS，后被圆子3次，旋风1次踢出
    2. ASU成员谴责圆子支持旋风的行为

    分析：本次战争令反革命者逐渐认识到革命者的行动已经开始但由于新年联欢会无暇顾及，更加确认ASU的起义行动成功

    思潮转变(2021.2.9)¶

    1. 21世纪著名心理学家，哲学家，抽象派艺术家，麻醉师，欧洲各皇室御用工程师，建筑学家，设计师——龙妈对ASU进行了有关本次起义事件的深度考究，大幅度降低了成员的起义之心，开始逐渐了解事件本质
    2. 紫灵决定与皮社友好合作，与ZPS社长旋风谈心，以和平的方式解决这次事件
    3. 伽企图再次发动起义，被龙妈禁言
    4. 晚上，紫灵，伽与圆子聊天试图劝说圆子加入ASU，得知圆子持中立态度
    5. 白朴建议紫灵与旋风谈心，但旋风的回复令人愤怒——虽知错在哪，但让下属处理好一切

    分析：若旋风谈心成功，或许ASU将与ZPS和谐相处

    独立日(2021.2.10)¶

    1. 白朴说服紫灵决定于下午ZPS新年联欢会结束后辞职
    2. 白朴与紫灵成功说服其他两名管理——神圣，云冲辞职
    3. 下午2点，ZPS新年联欢会开始
    4. 下午3点半，白妈向ZPS主群提交辞呈，并退出ZPS
    5. 10分钟后，紫灵，羽酱退出ZPS。神圣，云冲在ZPS观察现状
    6. 约5分钟后，旋风宣布解散ZPS

    分析：因为ZPS核心人员——后期，编剧，演员等职位的缺失，从而无法支持ZPS继续生存，被迫解散
